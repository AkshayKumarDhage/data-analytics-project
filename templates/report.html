Heart Disease Detection.¶

Introduction¶

The mini project “Heart Disease Detection” basically focuses on data analysis and making predictions based on the analysis using machine learning. Performing data analysis on people’s medical data with respect to heart health can help in finding insights and correlation of various factors with the heart health of an individual. The analysis shall be a great help in the medical industry which shall help in further improvements. Based on the analysis predictions shall be made to detect the presence of heart disease by using machine learning.

For the analysis, we are using Python programming language, as it supports all the required modern libraries to perform analysis.

Dataset overview,

Age - age in years

Sex - (1 = male; 0 = female)

Cp - chest pain type

Trestbps - resting blood pressure (in mm Hg on admission to the hospital)

Chol - serum cholesterol in mg/dl

Fbs - (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)

Restecg - resting electrocardiographic results

Thalach - maximum heart rate achieved

Exang - exercise induced angina (1 = yes; 0 = no)

Oldpeak - ST depression induced by exercise relative to rest

Slope - the slope of the peak exercise ST segment

Ca - number of major vessels (0-3) coloured by fluoroscopy

Thal - 3 = normal; 6 = fixed defect; 7 = reversable defect

Target – 1 (presence of heart disease) or 0 (no presence)

Here Target variable indicates the presence or absence of heart disease hence is classification objective, and we require to predict Target variable.

1.4. Data Pre-processing¶

I. Encoding on feature variables/Feature selection.¶

In the given dataset we find many of the feature variables which can take a few number of values like, “Sex” variable can have value 0 or 1. “cp” variable can have any of four values 1 or 2 or 3 or 4. “fbs” variable can have value as 0 or 1. “restecg” variable can have value as 0 or 1 or 2. “exang” variable can have value as 0 or 1. “slope” variable can have value as 0 or 1 or 2. “ca” variable can have value as 0 or 1 or 2 or 3 or 4. “thal” variable can have value as 0 or 1 or 2 or 3.

For better model training, we shall use dummy encoding method for pre-processing before using the data for training machine learning model.

Dummy Encoding (Boolean Indicator), we shall create new variables with each of the possible values as a column and having 1 as the value for a particular column and rest columns as 0.

III. Splitting Dataset into Training and Test Dataset.¶

It’s good to separate out our processed dataset into train and test set. We shall train our machine learning algorithm on train dataset and for validation we shall do predictions on test set and calculate the accuracy.

Here, we split dataset such that 67% data is for training machine learning algorithm and remaining 33% is for testing our machine learning model.

Here, X_train consists of part of dataset with all the feature variables except “target” column. y_train consists of “target” column for X_train.

And X_test is for test dataset and its “target” column is in y_test set.

Machine Learning Model¶

I. Training and Testing¶

Since we require to detect presence or absence of heart disease for a given input data, we conclude it as a Classification problem. And since we have only two possibility, we shall use Binary Classification.

Machine Learning Algorithm: Logistic Regression is a binary classification machine learning algorithm.

Logistic regression models the probabilities for classification problems with two possible outcomes. It’s an extension of the linear regression model for classification problems.

In [14]:
#training LogisticRegression model for classification 
from sklearn.linear_model import LogisticRegression 
from sklearn.metrics import accuracy_score 
lr = LogisticRegression() 
lr.fit(X_train, y_train) 

pred = lr.predict(X_test) 
print('Logistic Regression accuracy score: ', accuracy_score(y_test, pred)*100)

Logistic Regression accuracy score:  84.0

We use sklearn library in python for logistic regression model.

Logistic Regression model is trained on X_train, y_train dataset.

Based on the training, our machine learning model can be used for testing and prediction. Testing our model on X_test, y_test dataset.

We find that our Logistic Regression Model predicts with an accuracy of 84%.¶

With this good accuracy, we conclude that Logistic Regression Machine learning model has performed pretty good in classification for presence or absence of heart disease.
